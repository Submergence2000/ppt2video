# ppt2video
将PPT演示文稿转为视频以便于2倍速高效复习



## 使用条件：

**PPT为pptx格式且带有讲解未加密，操作系统为win10，office版本高于2010**



## 使用方法：

1. 将所有你想转换的PPT文件放在ppt文件夹中。

2. 运行程序main.exe，按照提示输入分辨率。
3. 输出的视频会自动放入video文件夹中。
